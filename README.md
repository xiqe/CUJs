# CUJs
基于ES6编写的常用JS函数


#### 项目架构

``` js
- CUJs/                 # 项目名称
  - src/                   # 开发目录
    + resources/           # 资源目录
        + less/                # css资源
        + img/                 # 图片资源
        + js/                  # js&jsx资源
    + components/           # 公共组建目录
    - format                # 格式化
  + dist/                # 编译输出目录
  + example/               # 示例
  webpack.config.js        # webpack配置文件
  package.json             # 项目配置
  README.md                # 项目说明
```
